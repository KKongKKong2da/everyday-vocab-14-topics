# Everyday Vocab: 14 Topics

**일상 영단어 14개 주제**를 한 장씩 학습하는 단일 HTML 웹앱입니다.

- 본문: 14개 주제, 3,360개 카드
- 부록 A: 국가·수도 색인, 197개 카드
- 전체: 3,557개 카드, 3,898개 예문
- 별도 서버나 빌드 과정 없이 `index.html`만으로 실행

## GitHub Pages

1. 이 저장소 루트에 `index.html`을 둡니다.
2. GitHub의 **Settings → Pages**에서 배포 브랜치를 선택합니다.
3. 생성된 Pages 주소로 접속합니다.

기본 전체 학습에서는 부록을 제외한 14개 주제만 표시합니다. 검색은 부록까지 포함하며, 필터에서 **부록 A · 국가·수도 색인**을 직접 선택할 수 있습니다.
